GRAMATICA Para guiarme

Simbolo inicial
E

Producciones:

E  -> T E'

E' -> opsuma T E'
E' -> ε

T  -> F T'

T' -> opmul F T'
T' -> ε

F  -> id
F  -> num
F  -> ( E )

Terminales

opsuma  = + | -
opmul   = * | /
id      = identificadores tipo variable
num     = numeros enteros
( )     = parentesis

Importante:

Esta gramatica define expresiones aritmeticas dnde

multiplicacion y division tienen mayor precedencia que suma y resta
las operaciones son asociativas por izquierda
Ejemplo:
a+b+c
es interpretado:
(a+b)+c
siempre.
